# Public Boundary

This repository is a bounded publication surface, not a private workspace, staging
area, or mirror of internal research.

Allowed content is limited to reviewed architectural prose, public citations,
sanitized static media, and the files explicitly named in the public-path allowlist.

The repository must not contain protected algorithms, solver geometry, operational
thresholds, decision tables, Witness implementations, private corpus fingerprints,
credentials, deployment data, or names and routes that reveal private repositories.

Every new path fails closed until it is reviewed and deliberately added to the exact
allowlist. Draft branches and pull requests are public and receive no confidentiality.
